# HealthQuest SMART Goals

## Overview

After completing the barrier assessment, users select ONE SMART goal from their top 3 barriers. Goals are age-appropriate and customizable.

**SMART = Specific, Measurable, Achievable, Relevant, Time-bound**

---

## Customizable Parameters

All goals have adjustable:
- **Duration:** 5/10/15/20/30/45/60 minutes (age-appropriate ranges)
- **Frequency:** Twice a week / Three times a week / Daily

---

## Data Structure (for implementation)

```lua
-- SMART Goal Template
SMARTGoal = {
    id = "mental_health_1",
    category = "mental_health",
    ageGroup = "6-12",
    template = "I will practice deep breathing exercises for {duration} minutes {frequency} to help manage my feelings.",
    durationOptions = {5, 10, 15},
    frequencyOptions = {"Twice a week", "Three times a week", "Daily"},
    defaults = {
        duration = 10,
        frequency = "Three times a week"
    }
}

-- User's Selected Goal
UserGoal = {
    goalId = "mental_health_1",
    category = "mental_health",
    duration = 10,
    frequency = "Three times a week",
    startDate = 1234567890,
    targetDays = 7,
    completedSessions = 0,
    assignedPath = "MindfulMoods"
}
```

|-----------|--------|
| Goal selected | 100 coins (assessment completion) |

| 3 sessions completed | 50 XP + 25 coins |
| Goal fully completed | 100 XP + 50 coins + badge |

## Parent/Caregiver Mode

Parents can:
1. Take assessment themselves OR
2. **Match child's barriers automatically**
3. Receive parallel goals to support their child
4. Track family progress together

Example Parent Goal (matching child's Mental Health barrier):
> "I will practice deep breathing exercises with my child for 10 minutes three times a week to help us both manage our feelings together."
